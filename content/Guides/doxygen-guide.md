
# Doxygen {#doxygen_usage}

To write custom doxygen pages, 
1. Go to the `content/Guide` folder
2. Create a file with `.md` extension
3. Write the documentation using Markdown format

To preview locally,
1. Go to root folder,
2. Install doxygen? should alr be installed if follow hlp1 and hlp2. I use 1.9.8, haven't tested for other versions
3. Run `doxygen` in cmd prompt. This will take some time (~3 mins?)
4. Open `docs/html/index.html` in browser

Format info:
- [Markdown](https://daringfireball.net/projects/markdown/)
- [Doxygen's markdown docs](https://www.doxygen.nl/manual/markdown.html)
