### Controls
| Action            | Primary Keys | Alternative Keys |
| ----------------- | ------------ | ---------------- |
| Move Left / Right | `←` / `→`    | `A / D`          |
| Jump              | `C`          | `<space>`        |
| Dash              | `Z`          | -                |
| Attack            | `X`          | -                |
| Ground slam       | `↓` + `X`    | `S + X`          |

Advanced:
- Wall Jump: *Jump* while touching a wall to perform a wall jump
- Combo attacks: Hold *Attack* key to execute combos
